## 四大构词类型

词根 - 单词的核心词义
前缀 - 改变词义
后缀 - 改变词性

|        构词                 |  
|:----------------------:|
| 前缀 + 词根             | 
| 词根 + 后缀             |
| 前缀 + 词根 + 后缀 |
| 单词 + 单词            |


- inter  两者之间的
	international: 国际的
	interpersonal: 人与人之间的
	interpersoanlrelation: 人际关系
	internet: 互联网
	interfere: 干涉
	intervene:  vene -> come 介入期间，干预
	interrelate: relate -> 关系。 相关关系
	interact: 相关影响
	interrupt: rupt-> break 中断，打断


- in-/ im-/ il-/ ir- 表示否定
	in-capable: 无能的
	in-advisable: 不明智的
	in-accurate: 不准确的
	in-proper: 不恰当的
	im-material: 非物质的
	im-mature: 不成熟的
	im-balance: 不平衡的
	il-legal: 非法的
	ir-regular: 非常规的
	ir-relevant: 不相关的

- dis- 表否定
	dis-appoint: 使失望
	dis-appear: 消失
	dis-approve： 不同意
	dis-able：使不能
	 


-ate 
	- 表形容词,具有...的
	- 表动词，做，造成
	appreciate 增值的，感激，欣赏
	depreciate 贬值的
- price 价格
	

- anti- 反、抗（aginst, opposite）
	anti-Japanese 抗日
	anti-Fascist 反法西斯者
	anti-missile 反导弹的
	anti-aircraft 防空的
